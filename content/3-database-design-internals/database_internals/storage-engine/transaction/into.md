> [!info] What is a Transaction ?
> A database transaction is a logical unit of work consisting of one or more data manipulation operations executed as a single, indivisible sequence. 
> 
> Within a Relational Database Management System, a transaction ensures that either all operations within the unit are executed successfully and permanently applied to the database, or none of them are.
> A database transaction has to preserve atomicity, consistency, isolation, and durability which is also known as ACID properties.
